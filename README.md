# README
This is a guide on how to setup the project and its requirements

## Requirements:
- Postgres
- Redis
- Ruby 2.5.1>


## Getting started

- Clone the project to your machine then run inside the folder:
- 'rails db:migrate'
- 'rails sb:seed'
- 'bundle exec sidekiq'
- 'rails server -b 127.0.0.1 -p 3000 -e development'
